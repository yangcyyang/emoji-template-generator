# 表情包模板生成器 - 版本记录

## 版本历史

---

### V1.0 - 原始版本 (index-simple.html)
**创建时间**: 2024-02-20
**特点**:
- 单文件HTML，无需后端
- 手动选择文件夹
- 手动输入标题/副标题/数量
- ColorThief自动提取背景色
- 4模板一键下载
- 内嵌SVG微信/QQ图标

**适用场景**: 少量表情包，追求最佳效果

---

### V2.0 - 增强版本 (index.html) ⭐当前稳定版
**创建时间**: 2024-02-18
**基于**: V1.0 + 功能增强

#### 新增功能:
1. **双模式支持**
   - 单文件夹模式（原版）
   - 批量集合模式（新增）

2. **智能标题解析**
   - 自动从文件夹名提取标题和副标题
   - 支持 `标题·副标题` 格式

3. **更多控制选项**
   - 主图选择器（可从文件夹选任意图片做主图）
   - 表情包排序方式（随机/顺序）
   - 图片数量显示

4. **批量处理能力**
   - 支持扫描上级文件夹下的多个子文件夹
   - 一键处理全部

5. **UI优化**
   - 更好的视觉样式
   - 添加Toast提示
   - 模态框大图预览
   - 响应式布局

6. **稳定性改进**
   - 更好的错误处理
   - 文件过滤（排除_key等）

**适用场景**: 手动处理，需要更多控制选项

---

### V3.0 - Python批量版 (auto_processor.py)
**创建时间**: 2024-02-19
**特点**:
- 纯Python实现，无需浏览器
- 全自动批量处理
- 自动打包成ZIP
- 支持自定义配置(config.json)
- 快速（1秒/套）

**缺点**:
- 头图布局有偏移（待修复）
- 颜色提取不如ColorThief精准

**适用场景**: 大量表情包自动化处理

---

### V4.0 - 浏览器自动化版 (auto_batch_v2.py)
**创建时间**: 2024-02-19
**状态**: 🚧 不稳定，开发中

**技术栈**:
- Python + Selenium
- 本地HTTP服务器
- 浏览器渲染 + html2canvas

**问题**:
- 运行极慢（>2分钟/套）
- 经常卡住
- 不适合生产使用

---

## 版本选择指南

| 需求 | 推荐版本 | 文件 |
|-----|---------|------|
| 快速批量处理 | V3.0 Python版 | auto_processor.py |
| 完美效果+手动 | V2.0 增强版 | index.html |
| 简单快速手动 | V1.0 原始版 | index-simple.html |
| 全自动化+完美效果 | 等待V4.0完善 | - |
